Tenses Made Easy To Grow Your English Writing and Speaking Skills.

Rules of Tenses help one understand how to correctly use the different tenses in a sentence, without making a grammatical mistake.

Present Tense Past Tense Future Tense

Present Tense

Present Tense can be defined as an expression for an activity that is currently in action or is habitually performed. It is used for a state that generally exists or is currently ongoing.

1-Simple Present Tense

Daily habits, facts, and general truths.
RULES
+ Sub + V1 (s/es) + Object
- Sub + do/does not + V1 + Object
? Do/Does + sub + V1 + Object?
Example
+ He reads books.
- He does not read books.
? Does he read books?

2-Present Continuous Tense

Actions happening right now.
RULES
+Sub + am/is/are + V-ing + Object
-Sub + am/is/are not + V-ing + Object
? Am/Is/Are + sub + V-ing + Object?
Example
+She is dancing.
-She is not dancing.
?Is she dancing?

3-Present Perfect Tense

Actions completed with a present result.
RULES
+Sub + has/have + V3 + Object
-Sub + has/have not + V3 + Object
?Has/Have + sub + V3 + Object ?
Example
+I have finished my work.
-I have not finished my work.
?Have I finished my work?

4-Present Perfect Continuous Tense

Started in the past and still continuing.
RULES
+Sub + have/has been + V-ing + Object
-Sub + have/has not been + V-ing + Object
?Have/Has + sub + been + V-ing + Object?
Example
+It has been raining.
-It has not been raining.
?Has it been raining?

Past Tense

Past Tense is used to express an action that happened or was completed in the past.

1-Simple Past Tense

Completed actions in the past.
RULES
+ Sub + V2 + Object
- Sub + did not + V1 + Object
? Did + sub + V1 + Object?
Example
+ She wrote a letter.
- She did not write a letter.
? Did she write a letter?

2-Past Continuous Tense

Actions that were happening at a specific time in the past.
RULES
+ Sub + was/were + V-ing + Object
- Sub + was/were not + V-ing + Object
? Was/Were + sub + V-ing + Object?
Example
+ They were playing football.
- They were not playing football.
? Were they playing football?

3-Past Perfect Tense

Actions completed with a present result.
RULES
+Sub + has/have + V3 + Object
-Sub + has/have not + V3 + Object
?Has/Have + sub + V3 + Object ?
Example
+I have finished my work.
-I have not finished my work.
?Have I finished my work?

4-Past Perfect Continuous Tense

Started in the past and still continuing.
RULES
+Sub + have/has been + V-ing + Object
-Sub + have/has not been + V-ing + Object
?Have/Has + sub + been + V-ing + Object?
Example
+It has been raining.
-It has not been raining.
?Has it been raining?

Future Tense

Future Tense is used to express actions that will happen in the future.

1-Simple Future Tense

Actions that will happen in the future.
RULES
+ Sub + will + V1 + Object
- Sub + will not + V1 + Object
? Will + sub + V1 + Object?
Example
+ I will help you.
- I will not help you.
? Will I help you?

2-Future Continuous Tense

Actions that will be in progress at a certain time in the future.
RULES
+ Sub + will be + V-ing + Object
- Sub + will not be + V-ing + Object
? Will + sub + be + V-ing + Object?
Example
+ She will be traveling tomorrow.
- She will not be traveling tomorrow.
? Will she be traveling tomorrow?

3-Future Perfect Tense

Action that will be completed before a specific time in the future.
RULES
+ Sub + will have + V3 + Object
- Sub + will not have + V3 + Object
? Will + sub + have + V3 + Object?
Example
+ They will have finished the work.
- They will not have finished the work.
? Will they have finished the work?

4-Future Perfect Continuous Tense

Action that will continue for some time in the future.
RULES
+ Sub + will have been + V-ing + Object
- Sub + will not have been + V-ing + Object
? Will + sub + have been + V-ing + Object?
Example
+ He will have been working here for five years.
- He will not have been working here for five years.
? Will he have been working here for five years?
